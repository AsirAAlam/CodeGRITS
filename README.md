# CodeVision

## Requirements

```
python==3.8.8
tobii-research==1.10.1
```

## Development

1. Click `Run` - `Run Plugin` to run the plugin and test it.
2. Open command line and run `./gradlew build` to build the plugin.
3. Get the plugin zip file in `build/distributions` folder.
